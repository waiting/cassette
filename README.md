Cassette是一款免费开源的账户密码管理软件，将帮助管理您日益增多的网络账户和密码。

Cassette不会访问网络，打消您的顾虑。采用成熟的AES256算法加密您的信息，保证您账户密码的安全。人性化的找回管理密码策略，即使忘记了管理密码也有补救的措施。十分便捷的快捷键呼出方式，方便您的使用。
